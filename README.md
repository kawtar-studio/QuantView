# QuantView

A professional financial market dashboard built with modern web technologies.

## 🚀 Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- Vite

### Backend
- FastAPI
- Python

### Database
- PostgreSQL

## Planned Features

- Live Market Dashboard
- Stock Search
- Company Profiles
- Financial Statements
- Interactive Charts
- Watchlist
- Portfolio Simulator
- Stock Screener
- Economic Dashboard
- Investment Research

⚠️ This project is currently under active development.
